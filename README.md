## 📖 About

An interactive **Quiz App** that tests your knowledge with multiple-choice questions — complete with a **progress bar**, instant feedback, and a **final score screen**.

---

## 🎮 Features

<table align="center">
<tr>
<td align="center">📊<br/><b>Progress Bar</b><br/>Tracks quiz completion</td>
<td align="center">⚡<br/><b>Instant Feedback</b><br/>Correct ✅ / Wrong ❌ highlight</td>
<td align="center">🏆<br/><b>Score Screen</b><br/>Final result with score</td>
<td align="center">🔄<br/><b>Retry Option</b><br/>Restart the quiz anytime</td>
</tr>
</table>

---

## 🔄 Quiz Flow

```
Start Quiz → Question Appears → Select Answer
     ↓
Instant Correct/Wrong Highlight → Progress Bar Updates
     ↓
Next Question → ... → Final Score Displayed 🏆
```
