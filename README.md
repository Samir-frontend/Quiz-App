🧠 Quiz App



📌 Overview


Quiz App is an interactive web application that allows users to test their knowledge through multiple-choice questions.  
It provides a simple and engaging quiz experience with instant results.



✨ Features


- ❓ Multiple Choice Questions
- ✅ Instant Answer Checking
- 📊 Score Calculation
- ⏱️ Interactive Quiz Experience
- 📱 Fully Responsive Design
- 🎨 Clean and User-Friendly Interface

  

🛠️ Tech Stack


- HTML5
- CSS3
- JavaScript
